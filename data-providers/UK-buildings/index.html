Please follow the Instructions below to get started

To use this data processor, you need an API key from OS Data Hub. Follow these steps to obtain your API key:

  1. Go to the OS Data Hub website: https://osdatahub.os.uk/.
  2. If you don't have an account, sign up by clicking the "Sign Up" button and following the registration process.
  3. Once logged in, navigate to the section where you can create a new project.
  4. Select the "OS NGD API â€“ Features" or a similarly named option when choosing the API products for your project.
  5. Fill in the details for your project, such as naming your project and providing a brief description.
  6. After creating your project with the selected API, the platform will generate an API key for you.
  7. Copy the API key to a secure location. You will need this key to use the data processor.
